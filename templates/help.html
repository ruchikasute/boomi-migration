Help Documentation

This tool leverages Boomi's Atmosphere APIs to retrieve details about processes and their configurations. Below are the core APIs used during extraction and evaluation.

Atmosphere API Details
Method Name URI Endpoint HTTP Method Credentials
Get All Processes https://api.boomi.com/api/rest/v1/:accountId/Process/query POST Username / Password
Get Process Configuration https://api.boomi.com/api/rest/v1/:accountId/Component/:componentId/export GET Username / Password
Get Component Details https://api.boomi.com/api/rest/v1/:accountId/Component/:componentId GET Username / Password
Authentication

All API calls require basic authentication using your Boomi credentials. 

Authorization: Basic (user/pass)

For more, refer to the official Boomi Atmosphere API documentation.
